Hobbydeed

I've lately been involved Mikko Hedman's project Hobbydeed, a web service aiming to create a toolbox for all kinds of groups to get organized. The technology stack consists mostly of AngularJS complemented by SASS and the Twitter Bootstrap framework in the front-end, and a Java EE/Spring based backend using an embedded Neo4J graph database.
